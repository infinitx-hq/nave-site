Build Your Formats

The leverage day. Teach your repeatable formats to an agent once — then run them forever. Read this, run the build action, then take the self-check at the bottom.

01 · What a format skill is

Taught once. Run forever.

A format skill is one repeatable content format — a spotlight reel, a hype carousel, a no-talk montage — taught to an agent once, in your voice, then run forever.

That's the single move that makes a month-in-a-day possible: you stop making content by hand and start running your own formats.

02 · The vocabulary

Agent, skill, command

Agent

Who. A crew member — your editor, your strategist, your designer.

Skill

What. An ability the agent loads — what it's trained on. Your format lives here.

Command

How. The order you type to run it — one line, plain English.

An agent is who; a skill is what they're trained on; a command is how you give the order. Keep these three straight and everything in the OS makes sense.

03 · Real worked examples

This isn't theory

Tiempo Soccer (a real client): its formats are already skills the OS runs by name — tiempo-spotlight (a conversational interview cutdown), tiempo-carousel-hype (the big-type rally carousel), tiempo-montage (the no-talk music montage), tiempo-parent-testimonial. Each one produces the same format, in Tiempo's voice, every run. Producing their month is: pick which formats, run them.

Aquafeel Solutions Arizona (another client): the same idea on a home-services brand — a drawn-by-hand explainer format, a proof/receipt format. Different industry, identical mechanic: name the format, teach it once, own it.

Your brand has 3–5 formats you make over and over. Each one should be a skill you run, not a thing you rebuild.

04 · Finding a format worth skilling

Skill what you already repeat

Don't copy a competitor. Don't pick the hardest one. Don't ask the model to invent one. The format worth turning into a skill is the one you already make over and over — that's where the compounding is.

Once your formats are skills, producing a month is just running your formats across your grid. The work moves up front, into the skill. After that, a month is execution, not invention.
